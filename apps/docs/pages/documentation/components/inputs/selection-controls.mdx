## Accessibility

`<Checkbox />` adheres to the [`checkbox` role requirements](https://www.w3.org/TR/wai-aria-1.2/#checkbox).
`<Radio />` adheres to the [`radio` role requirements](https://www.w3.org/TR/wai-aria-1.2/#radio).

Both components require an associated `label` element and `id` to describes the field.
